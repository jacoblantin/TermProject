Detailed Description

This class contains the constructor and methods that initialize the control loop variables and allow for the control algorithm to be run with a given Kp, the proportional gain constant.

◆ run()

control.CLPControl.run ( self)

This method is called repeatedly to run the control algorithm.

The method runs the control loop a set amount of times every 10 ms. The time and position is tracked over a range of 2000 ms, or 2 seconds, and stored in a list of tuples.
