Detailed Description

This file contains the main file, modified from basic_tasks.py, to run the tasks for the ME405 term project sequentially in a real-time scheduler.

Each task is modified as a generator file, and the main file in this task runs each task after one another. There is also an extra task that prevents the turret from running in a continuous loop.
